@shwilliam/gradient

Gradient is a service to generate gradients -- duh! A common use case for this is generating placeholder avatars based on a user's email address or username.

Example requests

  • /boop
  • /boop.png
  • /boop?type=jpeg
  • /boop?size=20
  • /boop/radial
  • /boop?variant=radial
  • /boop/radial?size=20,20
  • /boop/linear?size=100.ico
  • /boop/linear/20,20.svg

See the docs for more details.
